# Tertiary Infotech Landing Page

A high-tech, dark-themed landing page for **Tertiary Infotech** — an enterprise software company specializing in Training Management Systems, Learning Management Systems, and AI-Powered Solutions.

## Tech Stack

- **React 18** — Functional components with hooks
- **Vite** — Fast dev server and optimized builds
- **Tailwind CSS v3** — Utility-first styling
- **Framer Motion** — Smooth scroll reveals and hover animations
- **react-scroll** — Smooth scroll navigation between sections

## Features

- Dark theme with neon blue/cyan/purple accents
- Glassmorphism cards with backdrop blur
- Animated gradient background with floating orbs
- Portfolio dropdown banner in navigation
- Responsive mobile menu with accordion
- Contact form with client-side validation
- Scroll-triggered reveal animations
- Fully responsive (mobile-first)

## Sections

1. **Hero** — Animated headline with gradient text and dual CTAs
2. **About** — Company overview with stat cards
3. **Services** — TMS, LMS, and AI Solutions as interactive cards
4. **Why Choose Us** — 6-item feature grid
5. **Contact** — Glassmorphism form with validation and success state
6. **Footer** — 4-column layout with links and socials

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
